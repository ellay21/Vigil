# Vigil Mobile App

The official mobile interface for the Vigil Industrial Safety System. Built with Flutter, this app provides real-time monitoring, alerts, and control for industrial environments.

## üì± Features

*   **Dashboard:** Real-time visualization of sensor data (Temperature, Voltage, Vibration, Gas).
*   **Dynamic Charts:** Interactive history charts with auto-scaling and gradient visualization.
*   **Push Notifications:** Instant alerts for critical system states.
*   **Voice Assistant:** "Talk" to your system to get status updates hands-free.
*   **Dark Mode:** Optimized UI for low-light industrial environments.
*   **Multi-Language Support:** English and Amharic support.
